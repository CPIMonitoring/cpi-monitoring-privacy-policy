Setup

From install to first health check.

01

Add the extension

Install from the Chrome Web Store or Microsoft Edge Add-ons. No account, no sign-up screen.

02

Open any SAP Integration Suite page

A small CIM button appears at the bottom-right. It's already reading your last hour of messages.

03

Pick a tenant volume profile

A one-time setup screen tunes page sizes and lookback windows to your tenant's traffic. Takes ten seconds.

04

Click the button, anytime

Dashboard, monitoring, artifacts, queues, certificates — all there. Drag the button anywhere; it remembers.
