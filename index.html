Privacy Policy for CPI Monitoring Extension

Effective date: May 20, 2026

Overview

CPI Monitoring Extension is a browser extension for Microsoft Edge that helps users enhance SAP Cloud Integration monitoring by providing dashboards, message insights, artifact runtime overview, message journey analysis, idle iFlow analysis, and dependency insights.

Data Collection

This extension does not collect, sell, share, or transmit user data to third-party servers.

Local Browser Processing

The extension runs locally in the user's browser. It reads SAP Cloud Integration monitoring and runtime data only from the user's active SAP CPI tenant session.

SAP CPI Data

The extension may read SAP CPI monitoring information such as integration flow names, message statuses, runtime artifact details, certificate/JMS/dependency information, and message processing log metadata when the user is already signed in to their SAP CPI tenant.

No Credential Collection

The extension does not collect SAP usernames, passwords, OAuth tokens, or login credentials.

No Third-Party Sharing

The developer does not sell, rent, share, or transfer user data to third parties.

Storage

The extension may temporarily store cache data in the user's browser to improve performance. This data remains local to the user's browser and is not sent to the developer or external servers.

Permissions

The extension requests browser permissions only to provide its SAP CPI monitoring features. These permissions are used to read and display relevant SAP CPI monitoring data for the user.
